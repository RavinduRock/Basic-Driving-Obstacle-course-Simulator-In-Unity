# Basic-Driving-Obstacle-course-Simulator-In-Unity
🚗 Car Driving Simulator (Obstacle Dodge Challenge)

A simple but responsive 3D car driving simulator built in Unity, where players navigate a road while avoiding dynamic obstacles. The focus of this project is smooth vehicle control, reaction-based gameplay, and basic obstacle interaction mechanics.

🎮 Gameplay Overview

Players control a car driving along a straight road while dodging obstacles placed at intervals. The objective is simple:
Survive as long as possible without crashing.

The game tests:

Reaction time
Steering control
Spatial awareness
🧠 Core Features
🚗 Smooth car movement and steering system
🛣️ Infinite-style straight road layout
💥 Collision detection and crash handling
⚡ Lightweight performance-focused design
🎯 Controls
W / Up Arrow → Accelerate
A / Left Arrow → Steer Left
D / Right Arrow → Steer Right
S / Down Arrow → Brake / Slow down

🛠️ Built With
Unity
C# scripting
Unity Physics System
Basic 3D primitives/models
